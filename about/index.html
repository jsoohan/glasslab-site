About

Glass Lab

What this is

An evidence index for medical aesthetic procedures, organised as a matrix of indication × MoA cells. Every cell is grounded in (a) a brand-free PubMed search using MeSH and TIAB terms, and (b) Korean clinic procedure listings mapped to that MoA.

It is built so a reader can start from a concern (e.g. melasma, midface volume loss, axillary hyperhidrosis) and see — transparently — which treatment families have RCT / meta evidence behind them, which only have observational data, and which are essentially brand-driven without published clinical trials.

Why brand-free

If a clinic page lists "Thermage FLX", the underlying generic class is monopolar radiofrequency. PubMed indexes the generic class. So Glass Lab queries the generic class, and keeps the brand mapping in a separate, fully exposed table.

That separation is what lets readers see the difference between "well-studied MoA with many brands" and "popular brand with little MoA-level evidence".
